Клиент сайта AniLibria.TV для Windows 10
Перейти на сайт AniLibria.TV
Системные требования
Windows 10 версии 17134 и выше. Наличие установленного приложения App Installer для установки не из Windows Store.
Основные возможности
Доступ к обширному каталогу релизов с возможность фильтрации и сортировки по различным критериям. Встроенный онлайн видеоплеер для просмотра контента с возможностями перемотки, переходов а также проецирования видеоконтента на устройства поддерживающие эту технологию (телевизоры, проекторы, телеприставки и т.п.). Просмотр комментариев к релизам. Поддержка различных устройств от планшетов до Xbox One S/X с различными устройствами ввода.
Варианты установки
Есть несколько вариантов установки приложения:
  • Установка через Windows Store
  • Установка с помощью Anilibria Installer
  • Установка через сайт
  • Установка тестовых версий
Далее будет описан каждый из этих вариантов в подробностях.

 Чтобы установить приложение через Windows Store Вам достаточно будет перейти на страницу приложения и нажать на кнопку Установить/Get. Установка с помощью Anilibria Installer
 Вы можете установить приложение с помощью Anilibria Installer. Это специальная программа которая выполнит все необходимые шаги для установки за Вас. Все действия которые будут выполнены инсталлятором идентичны тем что описаны в разделе Установка через сайт. Также Anilibria Installer можно повторно запускать чтобы принудительно обновить приложение в случае если оно не обновляется автоматически (всякое бывает). Скачать инсталлятор можно по этой ссылке.
Установка через сайт

Установка через сайт требует выполнения больших действий чем в случае с Windows Store и также наличия Административных прав в операционной системе.

Шаг 1
Прежде всего необходимо включить возможность установки неопубликованных приложений. Чтобы её включить перейдите в настройки Windows (шестеренка в меню Пуск) далее перейдите в Обновление и безопастность и выберите раздел Для разработчиков. В этом разделе выберите пункт Неопубликованные приложения. Данное действие нужно выполнять только раз при первой установке приложения, после чего просто не меняйте эту настройку.

Шаг 2 Во время первоначальной установки и после истечения срока сертификата необходимо установить сертификат в доверенные корневые центры сертификации. Если у Вас возник вопрос зачем это нужно и что за сертификат Вы будете ставить в святая святых хранилища сертификатов то все очень просто. Установка сертификата гарантия от операционной системы что Вы не сможете установить "левый" софт просто так. Но после того как Вы добавите сертификат это гарантирует что Вы в здравом уме позволили какому-либо приложению быть установленным на Вашу систему. Чтобы установить сертификат необходимо на странице установки нажать на пункт Additional resources и кликнуть по ссылке Publisher Certificate для скачивания файла. Далее с помощью проводника (или любого другого аналогичного приложения) запустить его. После чего откроется диалог содержащий информацию о сертификате. Нажмите на кнопку Установить сертификат, далее выберите опцию Локальный компьютер после чего опцию Поместить все сертификаты в следующее хранилище и нажав на кнопку Обзор выберите пункт Доверенные корневые центры сертификации. Шаг 3 Ну теперь осталось только установить приложение. На странице установки кликните по кнопке Get the app.
После установки приложения его можно найти открыв меню пуск и набрав в поиске Anilibria.
