Anilibria приложение-клиент для Windows 10
Системные требования
Windows 10 версии 17134 и выше. Наличие установленного приложения App Installer.
Варианты установки
Есть несколько вариантов установки приложения:
  • Установка через Windows Store
  • Установка через сайт
  • Установка тестовых версий
Далее будет описан каждый из этих вариантов в подробностях.
Установка через Windows Store
 Чтобы установить приложение через Windows Store Вам достаточно будет перейти на страницу приложения и нажать на кнопку Установить/Get. Обновления будут автоматически применяться согласно настройкам системы, обычно автообновление включено по умолчанию. В данный момент приложение еще не опубликовано в Windows Store.
Установка через сайт

Установка через сайт требует выполнения больших действий чем в случае с Windows Store и также наличия Административных прав в операционной системе.

Шаг 1
Прежде всего необходимо включить возможность установки неопубликованных приложений. Чтобы её включить перейдите в настройки Windows (шестеренка в меню Пуск) далее перейдите в Обновление и безопастность и выберите раздел Для разработчиков. В этом разделе выберите пункт Неопубликованные приложения. Данное действие нужно выполнять только раз при первой установке приложения, после чего просто не меняйте эту настройку.

Шаг 2 Во время первоначальной установки и после истечения срока сертификата необходимо установить сертификат в доверенные корневые центры сертификации. Если у Вас возник вопрос зачем это нужно и что за сертификат Вы будете ставить в святая святых хранилища сертификатов то все очень просто. Установка сертификата гарантия от операционной системы что Вы не сможете установить "левый" софт просто так. Но после того как Вы добавите сертификат это гарантирует что Вы в здравом уме позволили какому-либо приложению быть установленным на Вашу систему. Чтобы установить сертификат необходимо на странице установки нажать на пункт Additional resources и кликнуть по ссылке Publisher Certificate для скачивания файла. Далее с помощью проводника (или любого другого аналогичного приложения) запустить его. После чего откроется диалог содержащий информацию о сертификате. Нажмите на кнопку Установить сертификат, далее выберите опцию Локальный компьютер после чего опцию Поместить все сертификаты в следующее хранилище и нажав на кнопку Обзор выберите пункт Доверенные корневые центры сертификации. Ниже видео инструкция.

Шаг 3 Ну теперь осталось только установить приложение. На странице установки кликните по кнопке Get the app.
После установки приложения его можно найти открыв меню пуск и набрав в поиске Anilibria.
